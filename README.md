# Ecommerce Analytics Dashboard

A modern and responsive e-commerce analytics dashboard built for real-world applications.

Designed with a clean "Bento Grid" layout, this dashboard focuses on clarity, usability, and visual hierarchy to make data easier to understand and act on.

---

## Features

* Interactive charts for sales, inventory, and customer data (powered by Chart.js)
* Dynamic filtering system for real-time data updates
* Customer activity tracking
* Inventory and product performance insights
* Notification panel for important updates
* Clean and modern dark UI with strong visual contrast

---

## Design Highlights

* Bento grid layout for better content organization
* Carefully selected color system for data visualization
* Responsive design for desktop and mobile
* Focus on readability and user experience

---

## Tech Stack

* React / Next.js
* TypeScript
* Tailwind CSS
* Chart.js

---

## Use Cases

* E-commerce dashboards
* SaaS admin panels
* Analytics platforms
* Internal tools
